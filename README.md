# Image Processing

## Description
The package image_processing_thayse is used to:

Processing:
- Histrogram matching
- Structural similarity
- Resize image

Utils:
- Read image
- Save image
- Plot image
- Plot result
- Plot histrogram

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install image_processing_thayse:

```bash
pip install image_processing_thayse
```
